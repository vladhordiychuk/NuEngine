üí° Why Vector2 is so much faster?

NuMath: Uses full __m128 SIMD register (128-bit) even for 2D vectors

GLM: Uses scalar float x, y (no SIMD for Vec2)

Result: NuMath processes Vec2 with the same SIMD efficiency as Vec4!
